# 애널리스트 총 추천 수 확장

## 변경 개요

- 총 추천 수가 10개 안팎에 머무르던 원인을 API 수집 범위에서 수정
- 화면은 기존처럼 전체 건수를 보여주고, 목록은 페이징으로 처리

## 원인

- 해외 애널리스트 데이터 수집이 `US_STOCKS.slice(0, 10)` 으로 잘려 있어 처음 10개 종목만 조회하고 있었음
- 국내 샘플 리포트 생성도 10개 종목만 사용해서 네이버 수집 실패 시 총 추천 수가 작게 고정되기 쉬웠음
- 네이버 리서치 수집도 1페이지 만 읽고 있어 최근 리포트 수가 적을 때 전체 건수가 부족해질 수 있었음

## 수정 내용

- `pages/api/analyst-reports.ts`
- 네이버 리서치 목록을 1페이지 고정 대신 1~3페이지까지 순회 수집
- 해외 애널리스트 데이터 수집에서 `slice(0, 10)` 제거
- 국내 샘플 리포트 종목 수를 20개로 확장

## 기대 효과

- 총 추천 수가 차트 TOP 10과 분리되어 실제 수집 건수 기준으로 증가
- 데이터가 많아질수록 애널리스트 추천 테이블은 페이지네이션으로 탐색 가능

## 검증

- `npm run build` 통과

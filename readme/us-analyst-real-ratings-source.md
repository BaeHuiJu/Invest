# 해외 애널리스트 실제 추천 이력 전환

## 변경 개요

- 해외 애널리스트 추천을 가상 브로커/가상 의견 생성 방식에서 실제 추천 이력 기반으로 전환
- JFrog(FROG)와 같은 실제 추천 케이스가 누락되지 않도록 해외 수집 로직 수정

## 원인

- 기존 `pages/api/analyst-reports.ts` 는 `US_STOCKS` 고정 목록의 시세만 조회한 뒤 브로커와 투자의견을 임의로 붙이고 있었음
- 따라서 실제 UBS, Radi Sultan, FROG 같은 추천 이력이 있어도 수집 구조상 반영될 수 없었음
- 특히 `FROG` 자체가 기존 해외 수집 대상 목록에 없었음

## 수정 내용

- `pages/api/analyst-reports.ts`
- 해외 추천 데이터 소스를 실제 `StockAnalysis` ratings 페이지로 변경
- 각 종목별 ratings 페이지에서 날짜, 증권사, 애널리스트, 신규 의견, 이전 의견, 목표주가를 파싱
- 현재가는 페이지 내 quote 데이터에서 파싱해 상승여력 계산
- 해외 대상 목록에 `FROG` 추가
- 기존 임의 브로커/임의 의견 생성 로직 제거

## 기대 효과

- UBS / Radi Sultan / FROG 같은 실제 공개 추천 이력이 기간 내라면 결과에 포함 가능
- 증권사 필터와 투자의견 필터가 실제 추천 이력 기준으로 동작

## 검증

- `npm run build` 통과

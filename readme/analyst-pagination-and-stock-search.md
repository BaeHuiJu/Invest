# 애널리스트 페이징 및 종목 검색 수정

## 변경 개요

- 애널리스트 추천 테이블에 페이지네이션 추가
- 총 추천 수는 전체 필터 결과 기준 유지
- 국내/해외 종목 목록에 검색 입력 추가

## 애널리스트 추천

- 대상 파일: `pages/index.tsx`
- `AnalystTab`에 `page` 상태와 `pageSize=10` 추가
- 테이블은 `filteredReports` 전체가 아니라 현재 페이지 데이터만 렌더링
- 필터 변경 시 페이지를 1로 초기화
- 전체 결과 수와 현재 표시 범위를 하단에 표시
- TOP 10 차트는 기존대로 `filteredReports.slice(0, 10)` 유지

## 종목 검색

- 대상 파일: `pages/index.tsx`
- `StockList`에 `query` 상태 추가
- 종목명 또는 티커 기준으로 클라이언트 필터링
- 검색 결과 수를 상단 카운트에 반영
- 검색 결과가 없을 때 별도 빈 상태 메시지 표시

## 원인 확인

- 국내 검색이 안 된 이유는 국내 API 오류가 아니라 `StockList`에 검색 UI와 검색 로직이 아예 없었기 때문
- 이번 수정으로 국내/해외 주식과 ETF 목록 모두 동일한 검색 동작을 사용

## 검증

- `npm run build` 통과
